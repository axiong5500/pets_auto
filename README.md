# PETS_AUTO

PETS全国英语登记官网自动化检测三级英语捡漏考场脚本
期盼大神的加入参与

## 代办功能

- 自动登陆
- 验证码自动OCR识别填充
- 考场实时检测
- 捡漏考场短信通知

## 技术

- java
- selenium
- webdrive
- 百度ocr
- 阿里sms
